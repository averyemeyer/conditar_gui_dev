## Suggested message

> We have an initial frontend prototype for conDitar. It supports protein plus
> reference-ligand and pocket-only inputs, exposes the current sampling options,
> and provides interactive 2D/3D review of generated molecules. The results view
> includes basic structure-derived metrics, compact dataset charts, candidate
> browsing, and downloads. It currently uses example outputs; backend execution
> will be connected next.

The interface and metrics shown are a development preview, not a completed
scientific analysis workflow.
